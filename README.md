## About

I am a second-year **B.Tech Computer Science Engineering** student who learns by building. Rather than consuming tutorials passively, I ship real-world projects end to end — from problem definition and architecture through implementation, testing, and deployment.

My work sits at the intersection of **Artificial Intelligence**, **Computer Vision**, and **modern web technologies**, with a growing interest in immersive AR/VR experiences delivered directly in the browser. Recent projects include real-time hand-tracking systems, offline voice assistants, and CNN-based emotion recognition pipelines.

I am currently deepening my foundations in **Data Structures & Algorithms**, **System Design**, **Backend Development**, **Docker**, and **AI Engineering**, with the goal of writing software that is not only functional but scalable and maintainable.

I am always open to collaborating on ambitious, technically interesting projects.

<details>
<summary><b>Enable the snake animation</b></summary>

<br/>

Create `.github/workflows/snake.yml` in this repository:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: Platane/snk@v3
        id: snake-gif
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/snake.svg
            dist/snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>
